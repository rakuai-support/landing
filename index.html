本当に5分で動画が完成しますか？

+

はい、プロンプト入力から動画完成まで平均3-5分です。AIが自動で音声生成・動画作成・QRコード生成を行うため、従来の翻訳・制作工程を大幅に短縮できます。

15言語すべてで同じクオリティですか？

+

高品質な音声合成により各言語で自然な発話を実現します。固有名詞や専門用語は表記ゆれが起きやすいため、台本上での表記確認をおすすめします。

動画の内容を後から修正できますか？

+

可能です。編集モードで台本修正→再生成が簡単にできます。QRコードはそのままで動画内容だけ更新されるため、印刷物の再作成は不要です。

小規模事業者でも導入できますか？

+

はい、個人店舗から大企業まで幅広くご利用いただいています。初回15コイン無料、クレジットカード登録不要でお試しいただけます。1分30円〜の低コストで本格的な多言語対応が可能です。

技術的な知識は必要ですか？

+

一切不要です。日本語で説明したい内容を入力するだけで、AIが自動で多言語動画とQRコードを生成します。パソコンやスマートフォンの基本操作ができれば、どなたでもご利用いただけます。

生成した動画の商用利用は可能ですか？

+

店舗案内や業務利用などの商用利用が可能です。アップロード素材の権利関係（画像・BGM・ロゴ等）はご利用者さま側でご確認ください。必要に応じてクレジット表記やライセンス条件に従ってください。

中国語の簡体字と繁体字の違いは？

+

簡体字は主に中国本土、繁体字は台湾・香港・マカオで使用されています。当システムでは両方に対応しており、ターゲット地域に応じて適切な文字・発音で動画を生成できます。

インストールや設定は必要ですか？

+

いいえ、一切不要です。Webブラウザがあれば即座にご利用いただけます。アプリのダウンロードや複雑な設定は必要ありません。登録後すぐに動画作成を開始できます。

どのような業界で活用されていますか？

+

飲食店、ホテル・旅館、小売店、教育機関、行政窓口など幅広い業界でご利用いただいています。特にインバウンド対応が必要な業界で高い効果を実感いただいています。

競合他社との違いは何ですか？

+

①圧倒的な低コスト（1分30円〜）②15言語完全対応③5分での高速生成④QRコード自動生成⑤クラウド一元管理など、インバウンド対応に特化した機能を総合的に提供している点が強みです。

QRコードの色やデザインは変更できますか？

+

はい、QRコードのカラーは変更可能です。日本の四季をテーマにしたカラーパレットや、ブランドカラーに合わせた色指定（RGBコード）にも対応しています。ロゴの掲載は現在対応していません。

音声のナレーターは選べますか？

+

はい、30種類の高品質ナレーター音声から選択可能です（明るい・落ち着き・重厚感など）。お店の雰囲気やターゲット層に最適な音声をお選びいただけます。

サポート体制はありますか？

+

はい、メールサポートにて平日10:00-17:00でご対応しています。操作方法から業界別活用方法まで、お気軽にお問い合わせください。導入支援も行っております。

コインの使い方は？

+

1コイン＝動画1分が目安です。秒数は切り上げで計算され、たとえば61秒の動画は2コインとなります。コイン残高は無期限で有効です。

領収書や請求書の発行はできますか？

+

領収書の発行機能は現在実装中です。発行希望の方はサポートまでご相談ください。

字幕やテロップには対応していますか？

+

ご利用中の言語に合わせた字幕を表示させることができます。字幕のタイミングは当社独自の計算方法を用いているため、まれに音声とのタイミングがずれる場合がございます。

動画の長さや文字数に制限はありますか？

+

文字数制限があります。目安として1本あたり3〜4分程度を推奨します。長くなる場合は複数本に分割すると視聴性が高まります。

営業時間や価格の変更時、すぐに全箇所を更新できますか？

+

同じQRのまま内容だけ差し替えられるため、張り替え不要で即時反映できます。季節メニューやイベント情報の切り替えにも向いています。

ユニバーサルデザインや情報アクセシビリティには配慮していますか？

+

「誰にとっても使いやすい」を前提に、色コントラスト・読みやすいフォント・シンプルな導線・音声中心の案内などに配慮しています。字幕機能提供後は、背景帯や文字サイズ等についてWCAG 2.1相当の推奨値をガイドとして提示予定です。

環境配慮やSDGsの観点でメリットはありますか？

+

QRで内容だけを差し替えられるため再印刷が減り、紙資源の削減に寄与します。また多言語の情報提供は、情報格差の縮小やインクルーシブな受け入れ体制（目標4/10/11/12）にもつながります。

各言語の文字化けやフォントの見え方は大丈夫ですか？

+

主要言語は必要に応じて Google の「Noto」ファミリー（Noto Sans JP／SC／TC／KR／Thai など）を自動割り当てしています。ラテン文字は Noto Sans を基本に OS 標準フォントへフォールバックするため、記号やダイアクリティカルマークも崩れにくい設計です。

どの端末・ブラウザでも視聴できますか？

+

最新のiOS/Android標準ブラウザと主要PCブラウザに対応しています。ご利用の端末やOSが古い場合は、一部機能が制限されることがあります。

QRコードはどのくらいのサイズ・場所に掲示すれば良いですか？

+

店頭やカウンターでは名刺～はがき大、動線上ではA5〜A4程度を推奨。背景とのコントラストを確保し、視線の高さ・手の届く位置に設置するとスキャン率が上がります。

画像・BGM・台本の著作権やライセンスはどうなりますか？

+

ご自身でアップロードする素材は、権利をお持ちのものをご使用ください。提供テンプレートやBGMは当社の作成したものですのでご自由にご利用ください。人物・施設の撮影素材は、必要に応じて肖像権・施設利用規約の確認をお願いします。

動画の公開範囲やセキュリティはどうなっていますか？

+

基本はURL公開での運用です。内容の差し替えは即時反映され、旧内容は表示されません。限定公開や店舗内だけでの運用については、運用設計（短期URL更新・QR差替え等）をご提案します。

BGMはどのように選べますか？

+

用途に合わせて多数のロイヤリティフリーBGMから選択できます（和風・ポップ・落ち着いた雰囲気など）。音量は自動調整され、動画に合わせて自然に再生されます。フェードイン／フェードアウトの細かい調整や、ご自身の音源アップロードには対応していませんが、利用シーンに適したBGMを簡単に選べるようになっています。

観光庁や自治体の補助金で導入費用をカバーできますか？

+

はい。観光庁の「観光振興事業費補助金（インバウンド受入環境整備高度化事業）」や、自治体が実施するDX推進・多言語対応の補助金を活用できる場合があります。これにより、導入費用の一部または大半をカバーでき、初期コストを抑えて運用を開始することが可能です。

視覚や聴覚に障害のある方への配慮として活用できますか？

+

はい。おもてなしQRメーカーは音声ガイド・字幕・多言語対応を備えており、ユニバーサルデザインの取り組みに適しています。視覚障害のある方には音声で、聴覚障害のある方には字幕で情報を伝えることができ、誰にとっても利用しやすい情報提供を実現できます。
